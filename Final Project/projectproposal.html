Project Proposal

What does it do?

A robotic arm that can be controlled using a phone or any other external means.
Each fingers would be able to move up and down independently.

Features

Each finger would be able to move independently base on the command given.
Robotic arm would be able to be controlled using a glove controller or your phone.

What will be made?

CAD Design, Graphics Everything in my project is designed on CAD
Laser cutting Servo holder
3D printing Casing for electronics
Micro-controller Custom made ATTINY1614 board
Input devices Phone, Robotic gloves
Output devices Fingers(servo) and Serial output(On your phone).
