What is this page about

This is to teach you how to program and test your ATtiny412 board from "Electronic Design".

Installing Boards Manager

Arduino IDE allows user to program their cores simply with a few downloads!
I would show you how I program my hello board 412 for this example.
Start by going to SpenceKonde's repository at github and going to installation.

Copy the link in the red circle.

Under preference, select "additional board URL" and paste the link in.
Arduino IDE would take a short while to download all files needed after.

To check if installation is done correctly, a new selection "megaTinyCore" should appear in your board manager.

Connections to board

Read up on the chip you are using and learn how to program it
(The photo above provides a clearer view of the chip than the one on their website)

You need to connect your FTDI to your UDPI before connecting it to your Helloboard.

If you are following my board, This is how it is suppose to be connected.

Burning bootloader

The new board would require a new bootloader to be burned before we are able to program the board.
You are require to burn a new bootloader if you change any setting on the board. The steps below shows the setting used on my board.

Firstly, ensure the board you selected is AT412. The programmer is set to SLOW 56600 baud and the clock speed to 8Mhz Internal.
open the images above to have a clearer view.

Once all the setting is completed. select burn bootloader.
